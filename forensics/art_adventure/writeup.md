For the keen-eyed folks, this is Piet! Piet is an esoteric language and can be interpreted.

Lets try to run the Piet-code, I use this [online npiet interpreter](https://www.bertnase.de/npiet/npiet-execute.php)

When you run this you will get two lines:
```
CnZhcj0nR1QxOHtDVlJHIFZGIFNIQSF9JwpwcmludCB2YXIK
Good number!
```
The first one is encoded is some matter, maybe base64.

lets try to decode it:
```
secret=CnZhcj0nR1QxOHtDVlJHIFZGIFNIQSF9JwpwcmludCB2YXIK
decoded = base64.b64decode(secret)
print decoded
```
This gives:
```
var='GT18{CVRG VF SHA!}'
print var

```

seems like `var` is rot13 (t=g and g=t)
Lets rotate back:
```
TG18{PIET IS FUN!}
```
